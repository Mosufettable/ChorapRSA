\[프로젝트명]: 단순 RSA방식 암호화/복호화 프로그램



\[설명]:

진짜로 작동하게끔 할 수 있는 RSA원리를 이용한 128비트 암호화/복호화가 가능한 프로그램입니다.



\[기능]:

제한이 거의 없는 변수 타입을 사용했고, mod연산과 수학적으로 더 일리가 있는 소수 판별법을 이용해서

암호화와 복호화를 해주는 프로그램입니다. 아직은 테스트 단계라서 사용자가 직접 입력하도록 구현하지는

않았지만 추후에 추가할 예정입니다.



\[비하인드]:

2025년 12월 경, 기말고사도 끝난 시점이였어서 집에 조금 보고 그냥 방치해둔 수학책(세상은 수학이다)

하나를 학교에 들고 가서 보고 있었습니다. 처음부터 쭉 읽고 있었는데 중간에 RSA의 원리가 나오더군요.

책에 방식이 그다지 또렷하게 써있지는 않았지만 너무 재밌던 나머지 방식을 찾아내서 제 공책에 열심히

적어놓고, 그걸 토대로 손으로 직접 연산도 해보았습니다. 그렇게 해서 이걸 프로그램으로 구현하고자 하는

욕심이 생겨서, 이 프로그램의 바로 전 프로그램인 첫 프로그램을 만들었습니다. 하지만, 일일이 제곱 연산을

하게끔 해놓았었고, 소수를 구하는 것도 많이 느렸던데다, 자료형의 한계가 너무 컸습니다(ulong 사용).

그래서 자료형을 biginteger로 바꾸고, chatgpt에게 물어봐가면서 더 나은 소수 판정법

(밀터-라빈 기법이였던 걸로 기억합니다.)과 mod연산을 적용했고, 마침내 암호화와 복호화 기능이 작동했습니다.

거기다가 128비트라는 거대한 수인데도 동작했고요. 앞으로 사용자 인터페이스도 개선하고 예외처리도 하려고

했었지만, rsa도 이미 C#의 라이브러리에 있다는 것을 알아버렸습니다. 그래서 거기까지는 포기했고 여기까지 만든

것을 이렇게 깃허브에 올려봅니다. 추후에 사용자 입력기능은 넣어보겠습니다.
